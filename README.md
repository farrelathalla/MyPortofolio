###

<p align="left">Install NodeJS and check if you have the latest version</p>

###

* npm
  ```sh
  npm install npm@latest -g
  ```

###

<h3 align="left">Installation</h3>

###

1. Clone the repo
   ```sh
   git clone https://github.com/farrelathalla/softeng.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the program
   ```sh
   npm run dev
   ```
